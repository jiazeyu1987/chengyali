使用说明

完成一次无形资产及长摊计提

按照以下步骤准备、校验并导出摊销结果。

  1. 下载标准模板，填写一级分类、名称、对应费用、原值、开始摊销月、入账月和摊销期限/月；残值可留空，留空按 0 处理。
  2. 选择计提月份并上传填写完成的 .xlsx 文件。
  3. 程序自动生成序号、月摊销额、累计月数、累计金额及期末净值。
  4. 已达到摊销期限的资产会封顶累计月数，累计摊销金额等于原值。
  5. 核对预览后导出结果，文件将保存到下载目录。
