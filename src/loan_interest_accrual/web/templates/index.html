使用说明

完成一次贷款利息计提

按照以下步骤准备、校验并导出计提结果。

  1. 优先下载标准模板，并按模板字段填写贷款主表和资金变动。
  2. 也可以上传已覆盖规则的历史台账，月份需与台账中实际数据一致。
  3. 选择计提月份，上传填写完成的 .xlsx 文件。
  4. 点击“计算并预览”，根据页面提示修正校验错误。
  5. 确认预览结果后点击“导出结果”，文件将保存到下载目录。
上传文件和计算结果仅用于本次操作，页面刷新后不保留。
